PPE pipeline detail:
  process(frame) -> publish to Redis stream "ppe" -> wait for result_timeout_sec
  -> remote GPU cluster consumes, runs YOLO, publishes back -> get_results
  -> on timeout: empty PipelineResult, logged as ppe_result_timeout
